# logscp_thrive
Requirements:
Bash - Windows users can use the Ubuntu for windows app.
sshpass


Using ubuntu app for windows, while in your root directory, run: ln -s /mnt/c/users/<InsertUserNameHere>/Documents

Then install this file in your Documents folder.

When connected to GRS VPN, run ./logscp.sh, it will prompt for the IP Address of the store, which day you need the log from, and the firefly password.
